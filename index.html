TempoIQ Overview¶

TempoIQ is an analytics backend for sensor data. Any connected device can write sensor data to TempoIQ via our write APIs. You can then perform historical analysis of the data through our analytics APIs.

TempoIQ’s data model makes it easy to map your real-world sensors to virtual data streams. A Device represents a single physical unit such as a power meter or vehicle, and each device has one or more Sensors attached. Each sensor corresponds to a single time-varying measurement on the device, such as voltage or temperature. This model allows for flexible queries on sensors within a single device, or comparing corresponding sensors across many devices.

Harness the power of TempoIQ in your application in a few easy steps:

  1. Model your devices and sensors
  2. Write sensor data
  3. Analyze the data
