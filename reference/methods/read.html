Reading Historical Data¶

Description¶

read(readRequest)¶
HTTP Endpoint:GET /v2/read/
Cursored:Yes

Reads data points from one or more devices and sensors.

Arguments:readRequest (ReadRequest) – Parameters for the read operation
Returns:ReadResponse with the requested data.
