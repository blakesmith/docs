Create device¶

createDevice(device)¶
HTTP Endpoint:POST /v2/devices
Arguments:device (Device) – The device to create
Returns:The Device that was created

Get device¶

getDevice(key)¶
HTTP Endpoint:GET /v2/devices/:key
Arguments:key (String) – The key of the device to get
Returns:The Device with the given key

Find devices¶

findDevices(search)¶
HTTP Endpoint:GET /v2/devices
Arguments:search (Search) –
Returns:A Cursor over the devices

Example:

{
  "search": {
    "select": "devices",
    "filters": {
      "device": {
        "attributes": {"building": "3"}
      }
    }
  }
}

Example¶

Return devices with keys dev32, dev33, and dev37. Note that in the HTTP API, the search object is wrapped in another object with an additional “find” field:

{
  "search": {
    "select": "devices",
    "filters": {
      "devices": {
        "or": [
          {"key": "dev32"},
          {"key": "dev33"},
          {"key": "dev37"},
        ]
      }
    },
  }
  "find": {
    "quantifier": "all"
  }
}

Update device¶

updateDevice(device)¶
HTTP Endpoint:PUT /v2/devices/:key
Arguments:device (Device) – The updated device
Returns:The updated Device

Updates a device with the provided metadata and sensors. To safely modify just some of a device’s properties, it is recommended to use this method in a GET-modify-PUT pattern. First, get the device object using getDevice or similar. Then, modify the metadata or sensors as desired. Finally, update the device on the server with this method.

A device’s key is immutable, so it is not possible to change a device key with this method. Calling updateDevice with a key that does not already exist in TempoIQ results in an error.

A device’s sensor configuration is currently also immutable.

Delete devices¶

deleteDevice(key)¶
HTTP Endpoint:DELETE /v2/devices/:key/
Arguments:key (String) – The key of the device to delete
Returns:Nothing
deleteDevices(search)¶
HTTP Endpoint:DELETE /v2/devices/
Arguments:search (Search) – Selector defining which devices to delete
Returns:The number of devices that were deleted
