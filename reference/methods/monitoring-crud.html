Create monitoring rule¶

createMonitor(rule)¶
HTTP Endpoint:POST /v2/monitors
Arguments:rule (MonitorRule) –
Returns:The created MonitorRule

Get monitoring rule¶

getMonitor(key)¶
HTTP Endpoint:GET /v2/monitors/:key
Arguments:key (String) – Key of the monitoring rule to get
Returns:The MonitorRule with the provided key

List monitoring rules¶

listMonitors()¶
HTTP Endpoint:GET /v2/monitors

Update monitoring rule¶

updateMonitor(rule)¶
HTTP Endpoint:PUT /v2/monitors/:key
Arguments:rule (MonitorRule) – The updated rule
Returns:The updated rule
setMonitorStatus(key, status)¶
HTTP Endpoint:PUT /v2/monitors/:key/status
Arguments:status (String) –

Delete monitoring rule¶

deleteMonitor(key)¶
HTTP Endpoint:DELETE /v2/monitors/:key
